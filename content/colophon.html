Typography

After an endless search I decided on TT Interphases Pro by TypeType as the typeface for this page. Something clean and elegant, yet readable and characterful.

Technical

This site was built using Hugo – a static site generator. It runs purely on HTML, SCSS and JavaScript.

Special libraries are Swup.js for page transitions, Emergence.js for scrolling animations and Vanilla Lazy Load for lazy image loading.
