Typography

Characterful and brutalist yet clean and approachable – Favorit by Dinamo combines “geometric rigidity with subtle oddities and a humorous touch” in a gorgeous way.

Technical

Built with Hugo using only HTML, SCSS, Markdown, and JavaScript. Deployed on Vercel.

Swup.js is used for page transitions, Lazyload for lightweight lazy image loading.

Photography

All images were taken or created by Linus Rogge, if not stated otherwise. I try to provide sources for all imagery on this site. Due to the nuances of the internet and platforms for visual inspiration, it's hard to always find the true author.
